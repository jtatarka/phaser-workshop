What is Phaser?

Phaser is a fun, free and fast 2D game framework for making HTML5 games for desktop and mobile Web browsers, supporting Canvas and WebGL rendering.

Game Design Fundamentals

Organization üéÆ

Games are broken up into multiple parts.

At the highest level:

  • Assets
  • Code

Assets

Assets are media files used in your game.

  • Designed by artists
  • Visual images
  • Music and sounds

Code

Writing a game requires three steps:

  1. Preload your assets
  2. Create your world
  3. Update on user input

Initialize It

All Phaser games start with Phaser.Game().


          var game = new Phaser.Game(800, 600, Phaser.AUTO, "", {
            preload: preload,
            create: create,
            update: update
          });

üì§ Preload

The preload() function is called first.

Use it to load your game assets.

üé® Create

Then create() is called next.

All of your assets are guaranteed to have loaded, so now you can use them in your game.

üîÅ Update

Finally, update() is called last.

It's called on a loop for the duration of our game so we can make things happen!

Sprites üéÆ

Geek speak for 2D graphics used in a game.

üìù Preload the game's background: sky.png.


            game.load.image("sky", "/assets/sky.png");

üìù Now, add it to the world.


            game.add.sprite(0, 0, "sky");

Spritesheets

A sprite, but with multiple frames in one file.

Physics üéÆ

Game objects have a body, or a set of properties which define how it moves and interacts with other objects in the game environment.

üìù Update the ground's body.


            ground.body.immovable = true;

üìù Add collision to the stars and platform.


            game.physics.arcade.collide(stars, platform);

Animations üéÆ

Animations bring life to your game.

üìù Add an animation for when the player moves to the right. üîé How is it different from "left"? 


            player.animations.add("right", [5, 6, 7, 8], true);

Now play the animation on user input.


            player.animations.play("right");
